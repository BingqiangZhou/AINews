# Density Guide

## 密度等级

### minimal（1-2 张）

**适用场景**：文章较短（< 800 字），或用户只想要最关键的视觉辅助。

**选择策略**：
- 只在文章的核心论点处插入 1 张
- 如果有明显的对比或框架，可加第 2 张
- 跳过所有过渡段和情感描写

**位置优先级**：
1. 核心概念解释处 → infographic 或 framework
2. 关键对比处 → comparison

---

### balanced（3-5 张）

**适用场景**：标准长度文章（800-1500 字），图文并茂但不过度。

**选择策略**：
- 每个主要段落（## 标题下）最多 1 张
- 优先选择信息密度高的段落
- 避免在相邻段落连续插图

**位置优先级**：
1. 核心概念 → infographic
2. 过程/流程 → flowchart
3. 对比分析 → comparison 或 framework
4. 数据展示 → infographic
5. 总结点 → framework 或 scene

---

### per-section（每个 ## 标题一张）

**适用场景**：结构化文章，每个章节都有独立主题。

**选择策略**：
- 每个 ## 标题下的第一段之后插入
- 根据该章节内容选择最合适的 type
- 如果章节内容纯叙事无可视化需求，可跳过

**Type 分配策略**：
- 概念解释章节 → infographic
- 步骤流程章节 → flowchart
- 对比分析章节 → comparison
- 框架模型章节 → framework
- 历史演进章节 → timeline
- 感悟叙事章节 → scene

---

### rich（6+ 张）

**适用场景**：教程、深度分析、长文（> 2000 字），需要丰富的视觉辅助。

**选择策略**：
- 在 balanced 基础上增加：
  - 文章开头的概念总览图
  - 每个关键术语的解释图
  - 文末的总结框架图
- 注意保持 type/style/palette 一致性
- 避免连续 2 张相同 type

---

## 密度与文章长度参考

| 文章长度 | 推荐 density | 预期插图数 |
|---------|-------------|-----------|
| < 500 字 | minimal | 1 张 |
| 500-800 字 | minimal | 1-2 张 |
| 800-1200 字 | balanced | 3-4 张 |
| 1200-1800 字 | balanced | 4-5 张 |
| 1800-2500 字 | per-section | 5-7 张 |
| > 2500 字 | rich | 7+ 张 |
